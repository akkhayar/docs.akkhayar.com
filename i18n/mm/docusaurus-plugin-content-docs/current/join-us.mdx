# လျှောက်ထားရန်

## သတ်မှတ်ချက်

လျှောက်ထားသူသည်
- မြန်မာနိုင်ငံသားဖြစ်သည်
- အင်္ဂလိပ်စကားပြောတတ်သည်

## အသိအမှတ်ပြု တက္ကသိုလ်များ

- GUSTO University, Yangon
- Device Business University, Yangon

ရန်ကုန်မှာရှိတဲ့ တက္ကသိုလ်အချို့ကို အသိအမှတ်ပြုပါသည်။ ပညာရေးစံနှုန်းနှင့် ကျောင်းသားဆက်ဆံရေးအပေါ် အခြေခံ၍ ဤစာရင်းကို ရွေးချယ်ထားပါသည်။ အကယ်၍ သင်သည် ဤတက္ကသိုလ်များအနက်မှ ကျောင်းသားတစ်ဦးဖြစ်ပါက အက္ခရာ မှ ကျောင်းနေဖက်များ တွေ့ရမည်ဖြစ်ပါသည်။ ဤတက္ကသိုလ်များမှ မဟုတ်ပါက စိတ်မပူပါနှင့်၊ လိုအပ်ချက်မဟုတ်ပါ။ အခြေခံသတ်မှတ်ချက်များနှင့် ပြည့်မီသရွေ့ လျှောက်ထားရန် တိုက်တွန်းပါသည်။
